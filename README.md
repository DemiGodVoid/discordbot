This is a regular discord bot, It also uses a gpt mindset running off
Microsoft model to help it give semi-accuate replies.
-
This github tool is being worked on, hoping to create a useful gpt bot that also has built in commands
so discord servers can have a bot they can actually chat with and at the same time, set their server up with.


Has built in insult messages and emotion messages to give it a better human-freind feeling.

How to talk to the bot? Simple.
-
Death message (Example: Hey, death! how are you?) is how you can talk to the bot as a friend.
-
death+insult message (Example: Death, could you insult me?) is how you can make the bot insult you.
-
Sad message (Example: Gosh, i feel sad) Will prompt the bot to give a meaningful reply back to you to help cheer you up.
